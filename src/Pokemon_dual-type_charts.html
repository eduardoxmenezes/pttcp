Pokémon dual-type charts

Chart info

The huge chart here shows the strengths and weaknesses of each dual-type, along with the number of Pokémon have that type. It looks complex but it's fairly straightforward to use - the type of the attack move is along the top row. Follow that down to the corresponding row to see how effective it is. Hover your mouse over a square to see a clear description.

The total number of Pokémon with each type are listed; click a number in that column to see a list of the actual Pokémon of that type combo. You may also like our unused type combos page.

Note: the number includes the same types switched, so Ghost/Dark and Dark/Ghost both show 2, because Sableye is Dark/Ghost, while Spiritomb is Ghost/Dark.

0 weaknesses

Prior to Generation 6, Ghost/Dark had zero weaknesses; with the introduction of Fairy this is no longer the case.

It should be noted however that the Eelektross family does have no weaknesses due to its ability, Levitate.
